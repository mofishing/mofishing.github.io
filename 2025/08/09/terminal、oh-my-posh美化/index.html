|总字数:376|阅读时长:1分钟|浏览量:

前置操作

安装windows terminal

从 微软商店（Microsoft Store） 当中安装即可。

安装地址（懒得搜点这里的链接）：Windows Terminal - Free download and install on Windows | Microsoft Store

安装 Oh my posh

如上，从微软商店安装。

安装地址（懒得搜点这里的链接）：oh-my-posh - Windows官方下载 | 微软应用商店 | Microsoft Store

安装 Powershell 7

相比于 Windows 默认提供的 Powershell，Powershell 7 拥有以下几个优势：

  1. 支持跨平台使用，增强了其灵活性和适用范围。
  2. 提供了更优越的性能表现。
  3. 处于持续更新状态，保证了功能的不断完善和安全性的提升。

简而言之，Powershell 7 是 Powershell 的升级版。

官网地址：

PowerShell 文档 - PowerShell | Microsoft Learn

Github 地址：

https://github.com/PowerShell/PowerShell

下载地址：

  1. GitHub - PowerShell/PowerShell: PowerShell for every system!
  2. 在 Windows 上安装 PowerShell - PowerShell | Microsoft Learn

选择合适自己的安装即可。

安装字体

oh-my-posh由于存在各种图标，推荐使用nerd-font字体，看自己的喜好下载即可

可以从 Nerd Fonts - Iconic font aggregator, glyphs/icons collection, & fonts patcher 官网下载

或者使用 scoop 的 scoop-nerd-fonts 下载

配置

PowerShell7 作为 Windows Terminal 的默认Shell

我一般使用 Listary 双击 Ctrl 唤出，此外还可以 win + r 输入 wt ，很推荐 Listary 作为日常启动软件、查找文件的工具，或许在速度上没有 everything 快，但在易用性上个人感觉更加舒适

下载网址：Listary – Free File Search Tool & App Launcher
